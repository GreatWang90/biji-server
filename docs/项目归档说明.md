## 3. 环境要求

- **Node.js**: >= 14.x
- **npm**: >= 6.x
- **浏览器**: Chrome / Firefox / Edge / Safari 现代版本
- **网络**: 局域网或互联网均可

## 4. 快速启动

```bash
# 1. 安装依赖
cd E:\biji-server
npm install

# 2. 启动服务
npm start
# 或
node server.js

# 3. 访问游戏
# 本机: http://localhost:3000
# 局域网: http://<本机IP>:3000
```

## 5. 部署方式

### 5.1 本地/局域网部署
直接运行 `node server.js`，同一网络内的设备通过局域网 IP 访问。

### 5.2 云平台部署 (Render.com)
1. 将代码推送至 GitHub 仓库
2. 在 Render.com 创建 Web Service，关联仓库
3. 构建命令: `npm install`
4. 启动命令: `node server.js`
5. 服务会自动使用 `process.env.PORT`

## 7. 迭代建议

以下功能可作为后续版本的改进方向：

1. **安全加固**: 密码 hash 存储、JWT 令牌认证
2. **服务端结算**: 将比牌结算逻辑移至服务端，彻底防作弊
3. **观战功能**: 允许旁观者观看对局
4. **聊天系统**: 房间内文字/表情聊天
5. **排行榜**: 金币/胜率排行
6. **断线恢复**: 游戏进行中断线后重连恢复手牌和状态
7. **数据库替代**: 使用 SQLite 或 MongoDB 替代 JSON 文件
8. **音效**: 发牌、出牌、胜利音效
